# Kurulum Adımları
Projeyi bilgisayarınıza indirin:
  
git  clone [proje-adresi]  
  
Proje klasörüne gidin:
  
cd [proje-adı]  
  
Gerekli paketleri yükleyin:
  
flutter  pub  get  
  
Uygulamayı başlatın:
  
 flutter  run
